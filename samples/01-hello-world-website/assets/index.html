🎉 Hello World!

AWS CDK で作った最初のウェブサイト

🚀 おめでとうございます！

このウェブサイトは、AWS CDK（Cloud Development Kit）を使って作成されました。 TypeScriptで書いたコードが、実際のAWSインフラストラクチャに変換されています。

🛠️ 使用技術

📦 Amazon S3

ウェブサイトファイルの保存

🌐 CloudFront

世界中への高速配信（CDN）

⚙️ AWS CDK

インフラをコードで管理

📚 次のステップ

  1. このウェブサイトのHTML/CSSを編集してみる
  2. CDKコードのコメントを読んでTypeScriptを理解する
  3. Sample 02（API Server）に挑戦する

💡 学習ポイント

  • S3静的ウェブサイト: HTMLファイルを置くだけでウェブサイト公開
  • CloudFront: 世界中のユーザーに高速でコンテンツ配信
  • CDK: インフラをコードで管理（Infrastructure as Code）
  • TypeScript: 型安全なJavaScript、IDEの支援が強力
