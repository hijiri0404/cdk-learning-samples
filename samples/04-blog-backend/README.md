# 📝 Sample 04: Blog Backend System

## 📖 概要
**開発状況**: 🚧 開発中（80%完成）

RDS PostgreSQL、VPC、Lambda を使用した本格的なブログバックエンドシステムです。

### 🎯 学習目標
- RDS（リレーショナルデータベース）の基本
- VPC（ネットワーク）の設計
- データベース接続とORM使用
- セキュリティグループの設定

### 💰 コスト見積もり
- RDS db.t3.micro: ~$12/月
- VPC NAT Gateway: ~$32/月
- Lambda実行: ~$1/月
- **合計: ~$45/月**
